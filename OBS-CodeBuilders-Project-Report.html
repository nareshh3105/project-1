Section 01

Project Overview & Objectives

3
What We Built

The OBS CodeBuilders UI Kit is a production-ready Figma design system that replicates the complete OBS Studio desktop broadcasting interface, reimagined with a neon purple → magenta "Code Builders" developer aesthetic. The kit is delivered as a fully interactive Figma prototype with every button wired to a navigation destination.

The entire kit was generated and maintained by a custom Figma plugin ("OBS CodeBuilders UI Kit Builder"), which creates, polishes, and wires all design elements programmatically — ensuring pixel-perfect consistency and zero manual drift.

🎯 Client Brief Summary
Build a complete Figma UI kit for an OBS Studio-inspired interface — every dockable panel, modal dialog, context menu, toolbar, and interactive state — themed with a neon gradient developer brand. The kit must be component-driven, token-based, and fully clickable in Figma's prototype mode.
Key Objectives
🎨 Design System
Full token suite (colors, typography, spacing, shadows) exported as tokens.json. All components reference tokens — zero hardcoded values.
🧩 Component Library
60+ reusable components covering every OBS interface element: panels, buttons, inputs, meters, menus, modals, and list items.
🖥 Assembled App Frames
Three full 1440×900 desktop frames showing Default, Studio Mode, and Floating Windows layouts — each a realistic, pixel-accurate mockup.
⚡ Clickable Prototype
980+ prototype links wired via Figma reactions API. Every button, menu, toolbar icon, and modal is navigable in the prototype player.
🔌 Plugin Automation
A custom Figma plugin builds the entire kit on demand, ensuring repeatable, consistent output with one click.
📦 Token Export
tokens.json in W3C DTCG format — ready for Tokens Studio, Style Dictionary, or any design-to-code pipeline.
Technology Stack
Figma Plugin API JavaScript (ES2020) Auto Layout Figma Variables / Tokens Figma Reactions API Inter Typeface JetBrains Mono Lucide Icons (SVG) W3C DTCG tokens.json

Section 08

Figma Plugin — Automation & Delivery

10
What the Plugin Does

The OBS CodeBuilders UI Kit Builder is a custom Figma plugin (manifest: obs-codebuilders-uikit) that automates the entire design system. Instead of manually placing every frame, it generates, polishes, and wires all components programmatically — ensuring zero drift between runs and complete reproducibility.

Plugin Buttons — What Each Does
ButtonWhat It DoesOutput
Build UI Kit Full build from scratch: creates all 8 pages, all components, all app frames, tokens, icons, modals, menus. Complete ~5 min run. Entire Figma file
Wire Prototype (7 flows) Wires the 7 core navigation flows between the 3 main app frames using Figma reactions API with DISSOLVE + MOVE_IN transitions. 7 prototype links
Wire Modal Prototypes Wires 10 modal/overlay scenes including settings, streaming confirmation, recording, virtual camera, and all menu overlays with scrim dismiss. ~10 modal links
Build Feature Components Builds 9 OBS feature area frames (Webcam Only, Screen Share, Intermission, Source Filters, Properties, Scene Collection, Profile Manager, Multiview, Auto-Config Wizard) on Page 06. 9 feature frames
Full Design Polish Enhances all app frames: fills preview canvas with gradient + desktop mockup, draws toolbar icon shapes, adds scene thumbnails with color badges, polishes audio meter segments, enriches status bar metrics, adds empty-state placeholders. 199+ polished elements
Build Advanced Flows Wires 116 advanced prototype links: settings tab navigation (7 tabs), add-source grid (10 source tiles → config scenes), streaming/recording/virtual-camera modal chains. 116 links
⚡ Wire All Buttons Comprehensive button wiring across all Page 03 frames. Explicit named wires + per-frame control panel loops + per-frame toolbar loops + fallback sweep (catches any unnamed interactive elements). Self-loop guard prevents a button wiring to its own parent frame. 981 wired (88 via sweep)
Export Design PDF Exports every named frame as a 0.5× PNG screenshot (via Figma exportAsync), encodes to base64, and downloads a complete HTML→PDF document with screenshots and descriptions of every section. Downloadable PDF
